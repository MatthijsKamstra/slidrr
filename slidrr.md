## code blocks

inline `code` or code-block:

```
class GuineaPig {
	var name : String;
	var age : Int;
	public function new(name, age) {
		this.name = name;
		this.age = age;
	}
}
var pig1 = new GuineaPig("Maisie", 1);
var pig2 = new GuineaPig("Tilly", 2);
```
